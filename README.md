## üì± About SpeakSummarize

SpeakSummarize is an AI-powered voice recording and transcription iOS app featuring Echo, your calm AI memory companion.

**Key Features:**
- Voice transcription using OpenAI Whisper
- AI-generated structured summaries
- Speaker diarization
- Semantic search across recordings
- Multi-chat conversations with Echo
- PDF export
